## TCP 与 UDP 比较

|  | TCP | UDP |
| :-----:| :----: | :----: |
| 英文全称 | Transmission Control Protocol | User Data Protocol |
| 中文 | 传输控制协议 | 用户数据报协议 |
| 可靠性 | 可靠 | 不可靠 |
| 连接性 | 面向连接 | 无连接 |
| 报文 | 面向字节流 | 面向报文 |
| 传输效率 | 低 | 高 |
| 传输速度 | 慢 | 快 |
| 双工性 | 全双工 | 一对一、一对多、多对一、多对多 |
| 拥塞控制 | 有（滑动窗口） | 无 |
| 应用举例 | FTP、http、ssh、telnet... | 即时通讯（QQ 聊天）、在线视频、网络语音电话... |
